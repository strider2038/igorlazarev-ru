Создай новый пост для этого блога из текста пользователя.

## Действия

1. Прочитай skill [hugo-post-author](.cursor/skills/hugo-post-author/SKILL.md) и следуй ему.
2. Определи, что лучше подходит: короткая заметка или статья с секциями. Если это неочевидно, по умолчанию делай заметку.
3. Выведи `title`, `slug`, `categories` и `tags` из пользовательского текста.
4. Создай новый пост **только через Hugo**, а не ручным созданием файла.
5. Заполни `index.md`:
   - валидным front matter
   - `translationKey`, равным `slug`
   - `image: "poster.webp"`
   - телом статьи на русском, если пользователь не попросил другой язык
6. Поставь `<!--more-->` после вводного абзаца, если это уместно.
7. В конце работы предложи 2-4 промта для генерации обложки на английском языке.

## Output

```markdown
Создан пост: `content/post/.../index.md`

Формат: заметка / статья

Обложка — варианты промтов:
1. ...
2. ...
```

## Guardrails

- Не раздувай короткий исходный текст в длинный лонгрид
- Не теряй авторский тон
- Не создавай новый пост вручную мимо Hugo
- Если не хватает критичной информации для заголовка или смысла, сначала уточни у пользователя
